Interface BeautyEffectOptions

美颜选项，用于 [setBeautyEffect]{@link ILocalStream.setBeautyEffect}。

Properties

Optional lighteningContrastLevel

lighteningContrastLevel: 0 | 1 | 2

亮度明暗对比度，与 lighteningLevel 参数搭配使用。

取值范围为 0、1 或 2。默认值为 1（原始对比度）。取值越大，明暗对比越强烈。

Optional lighteningLevel

lighteningLevel: undefined | number

亮度。

取值范围为 [0.0, 1.0]，其中 0.0 表示原始亮度，默认值为 0.7。可用来实现美白等视觉效果。

Optional rednessLevel

rednessLevel: undefined | number

红色度。

取值范围为 [0.0, 1.0]，其中 0.0 表示原始红色度，默认值为 0.1。可用来实现红润肤色等视觉效果。

Optional smoothnessLevel

smoothnessLevel: undefined | number

平滑度。

取值范围为 [0.0, 1.0]，其中 0.0 表示原始平滑等级，默认值为 0.5。可用来实现祛痘、磨皮等视觉效果。
